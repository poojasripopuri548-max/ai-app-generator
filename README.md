# 🚀 AI App Generator

An AI-powered web application that enables users to generate dynamic applications, forms, and dashboards using natural language prompts. The platform converts user requirements into structured UI components and interactive interfaces, providing a fast and intuitive app-building experience.

🌐 **Live Demo:** https://ai-app-generator-taupe.vercel.app/

---

## 📌 Overview

AI App Generator is a modern full-stack web application inspired by platforms like Base44. It allows users to describe the application they want in plain English, and the system generates interactive forms and dashboards dynamically.

The goal of this project is to simplify application creation while demonstrating modern full-stack development and AI integration.

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/poojasripopuri548-max/ai-app-generator.git
```

Navigate to the project

```bash
cd ai-app-generator
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Open

```
http://localhost:3000
```

## 🎯 Workflow

1. Enter a prompt describing the application.
2. AI processes the request.
3. JSON schema is generated.
4. Dynamic UI is rendered.
5. User interacts with the generated application.
6. Data is stored and displayed on the dashboard.
